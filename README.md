# Finance Dashboard

This is a simple finance dashboard built using React.
It helps users track their income and expenses and understand their spending in a clear way.

## Features

* Dashboard with total balance, income, and expenses
* Line chart to show balance over time
* Pie chart to show spending by category
* Transactions list with search and filters
* Add, edit, and delete transactions (admin only)
* Role switch (admin / viewer)
* Insights like top spending category and monthly comparison
* Data saved in localStorage
* Export data as CSV and JSON

## Tech Used

* React
* Recharts
* CSS

## How to Run

1. Install dependencies
   npm install

2. Start the app
   npm start

## Notes

* This project uses mock data (no backend)
* Role switching is handled on the frontend
* Focus was on clean UI and simple user experience

---

This project was built to practice frontend development and handling data in a simple dashboard.
